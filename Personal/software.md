## PyCharm
### 修改背景色
* 修改高亮所有相同字符串的背景色：
File -> Settings -> Editor -> Colors & Fonts -> General 
在Settings对话框的右侧选择: 
Code -> Identifier under caret 随后修改Background值

* 修改选中字符串的背景色
File -> Settings -> Editor -> Colors & Fonts -> General 
在Settings对话框的右侧选择: 
Editor -> Selection background 随后修改Background值
